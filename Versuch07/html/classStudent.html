bool Student::operator< ( const Student  student) const

Overloaded 'Less than' operator.

This overloaded operator serves the same binary function as "<" in C++. One student is "less" than the other if his/her MatNr is less than that of the student given as a parameter. For example, let Student u.matNr = 1000 and Student v.matNr = 2000. Then u < v -—> True v < u -—> False
