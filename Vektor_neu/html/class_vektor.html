Vektor Vektor::add ( const Vektor &  input) const

Addition of two vectors.

This method calculates the sum of two given vectors. Let v and w be summands. To calculate v+w this method has to be called in the form: v.add(w) or w.add(v) as addition operation is commutative in the vector space R^3.

◆ angle()

double Vektor::angle ( const Vektor &  input) const

Calculates the angle between two vectors in deg.

This method calculates the angle between two input vectors. It uses the fact that the dot product divided by the sum of lenghts of two vectors is the cosine of the angle between them. Then the arccos function provided in the cmath library is applied to get the angle and the radian output is converted into degrees

◆ ortho()

bool Vektor::ortho ( const Vektor &  input) const

Checks if vectors are orthogonal.

This method checks if the given vectors are orthogonal to one another, that is - is the angle between them = 90 deg? This requires the calculation of the dot product and when it's equal to 0 then the conclusion could be made that the input vectors are orthogonal. As calculations with floating point arithmetic are not 100% accurate, an EPSILON preprocessor constant is checked as a small tolerance value.

Parameters
inputThe 2nd vector
Returns
true if vectors are orthogonal, false otherwise
