double Add::evaluate ( ) const
virtual

Adds the expressions together.

This function adds the left and the right expressions together. They may be simple doubles or long expressions of the Expression datatype. This is why evaluation of both operands is needed before the addition operation can be performed, as to make sure to complete all the previous operations first.

Returns
The sum of the left and the right expressions
