double Sub::evaluate ( ) const
virtual

Subtracts the expressions.

This function subtracts the right expression from the and the left one. They may be simple doubles or long expressions of the Expression datatype. This is why evaluation of both operands is needed before the subtraction operation can be performed, as to make sure to complete all the previous operations first.

Returns
